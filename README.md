### Bash script that downloads the kernel source from https://kernel.org, compiles the kernel and loads it into the system. 
#### Script installation :
`git clone https://github.com/curar/auto-kernel-download-compile.git`
#### Turning on :
`./auto-kernel-1.0.sh`
